# wentao.gg

Personal portfolio site for [Wentao He](https://wentao.gg) — Data Engineer at Meta, with a background in Robotics (AI) from UPenn and Mechanical Engineering from CMU.

## Live

🔗 [wentao.gg](https://wentao.gg)

## Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) with App Router
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Effects:** WebGL Fluid simulation
- **Analytics:** Vercel Analytics + Speed Insights
- **Deployment:** Vercel

## Features

- Dark/light theme toggle
- Interactive WebGL fluid background effects
- Scroll-driven animations and parallax
- Responsive, mobile-first design
- Collapsible sections for projects and experience
- GitHub commit heatmap
- Dynamic code-splitting for below-fold components

## Projects

| Project | Description |
|---------|-------------|
| **🐋 Tracker** | Hyperliquid whale leaderboard tracking top traders by PnL, ROI, and volume with live API data |
| **PowerOPPS** | Powerlifting index calculator across 5 standardized scoring systems (IPF GL, DOTS, Wilks 2.0, IPF, Old Wilks) with reverse calculation |
| **ProgDash** | Google Sheets-powered training log with OAuth authentication for tracking powerlifting programming |
| **What's my RPE?** | *(Coming soon)* Velocity-based training tool using pose estimation to predict RPE and estimate 1RM from video |

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
